# AI Internship Projects

Welcome to the AI Internship Projects repository! This repository contains all the tasks and projects completed during the AI internship, focusing on various machine learning and deep learning concepts.

## 📂 Repository Structure

The repository is organized into three main tasks, each contained within its own directory:

### [Task 1: Customer Churn Classification](./Task1)
An end-to-end machine learning project that analyzes customer churn data to predict customer attrition.
- **Key Concepts:** Data exploration, preprocessing, classification models, model evaluation.
- **Technologies:** Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn.

### [Task 2: CIFAR-10 Image Classification with MobileNetV2](./Task2)
A deep learning project demonstrating how to train a MobileNetV2 model for image classification on the CIFAR-10 dataset.
- **Key Concepts:** Convolutional Neural Networks (CNNs), Transfer Learning, Image Inference.
- **Technologies:** TensorFlow, Keras, NumPy, Matplotlib, Pillow.

### [Task 3: Model Fairness, Bias, and Explainability Analysis](./Task3)
An advanced analysis of machine learning model fairness, focusing on identifying biases and explaining model predictions.
- **Key Concepts:** Explainable AI (XAI), Model Bias Mitigation (SMOTE), Global & Local Explanations.
- **Technologies:** SHAP, LIME, Imbalanced-Learn, Pandas, Scikit-Learn.

## 🚀 Getting Started

Each task is entirely self-contained and has its own `README.md` file and `requirements.txt`. 

To run any of the projects, simply navigate to the specific task folder, install the required dependencies, and execute the code.

**Example for Task 1:**
```bash
# Navigate to the task directory
cd Task1

# (Optional but recommended) Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use: .\venv\Scripts\activate

# Install the required modules
pip install -r requirements.txt
```

*Please refer to the `README.md` inside each task folder for detailed, step-by-step instructions on how to run the notebooks and scripts.*
